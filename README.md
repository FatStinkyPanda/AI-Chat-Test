## Architecture Overview

### 7 Advanced Cognitive Systems

1. **Causal Discovery Engine** - Discovers cause-effect relationships
2. **Multi-Step Inference Engine** - Chains reasoning to deep conclusions
3. **Conversation Predictor** - Predicts conversation trajectory
4. **Attention Mechanism** - Intelligently selects relevant information
5. **Knowledge Graph** - Extracts entities, relationships, and facts
6. **Meta-Learning System** - Learns optimal learning strategies
7. **Transfer Learning** - Applies patterns across contexts

### 26-Step Cognitive Pipeline

Every input triggers sophisticated processing:
- Perception & encoding
- Memory storage & retrieval with attention
- Knowledge extraction & causal discovery
- Multi-hop reasoning & inference chains
- Conversation prediction & strategy selection
- Autonomous deliberation (AI decides when ready)
- Response generation & learning

### Brain-Inspired Memory

- **Nodes:** Memories with content, embeddings, emotions
- **Edges:** 8 types (semantic, causal, emotional, temporal, etc.)
- **Activation:** Spreading activation like neurons
- **Working Memory:** Recent context (like human short-term memory)
- **Long-Term Storage:** Persistent across all sessions

## Commands

In the chat interface:

- `help` - Show available commands
- `stats` - Show system statistics
- `memory` - Show memory statistics
- `graph` - Show brain graph statistics
- `save` - Save current state
- `reset` - Start new conversation (keeps memory)
- `clear` - Clear screen
- `exit` / `quit` - Exit system
